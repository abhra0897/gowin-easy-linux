To verify license files having the TRL formatted keys on Linux, Suse and Windows platforms, sssverify-trl[.exe] should be used. Using sssverify executable may result in incorrect output.

Example of a sample TRL key (The SIGN="XXX" field is additionally present):

INCREMENT F1 snpslmd 2014.06 02-aug-2020 8 ISSUED=25-apr-2019 ck=175 \
	SN=TK:367258:660677 START=25-apr-2014 AUTH={ snpslmd=( \
	LK=3CFEC0B8E58FC1A4 SIGN="04AE 5414 E696 BB55 4D90 449B 7C02 \
	1BF7 4A4E C65C BFE4 200C A8E1 50F9 2055 13CF 0164 FAA3 2C78 \
	CF4C 5A89 5D52 90CC D91A E11A 0ED1 D873 1F81 6C34 F4EC") }
