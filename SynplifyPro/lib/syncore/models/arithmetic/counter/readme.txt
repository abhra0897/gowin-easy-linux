
The SYNCORE Counter compiler helps designers create counter models for their designs. 
These Counter models are written out in HDL and can be synthesized as well as simulated.
A testbench is provided for the Counter models created by the SYNCORE Counter compiler. 

For detailed information on the SYNCORE Counter compiler and its various options, click 
the COUNTER Info button at the bottom of this window.
