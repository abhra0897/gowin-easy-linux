SYNCore Byte-Enable RAM Model

The SYNCore byte-enable RAM compiler helps you create byte-enable RAM models for your designs. These models are written out in SystemVerilog and can be synthesized as well as simulated. A testbench is generated for this purpose.

For more information about the SYNCore byte-enable RAM compiler, refer to the following:

  • The built-in Byte-Enable RAM Compiler document, which you can access from the BYTE ENABLE RAM Info button.

  • The tools online help, where you can access information for the following from the online help:
