SYNCore

SYNCore is the Synplify IP Core Wizard. This tool simplifies the building of HDL models for IP blocks. The Verilog models it generates can be synthesized and simulated.

To get started, double-click the kind of model you want to create. This starts the wizard, which takes you through what you need to do to generate the model you want.

For detailed information about the models, use the following:

  • Use Info buttons in the respective core wizard
  • Synopsys online help for the synthesis tool

For other questions and issues, please contact support at www.solvnet.com .
