SYNCore FIFO Model

The SYNCore FIFO compiler helps you create FIFO models for your designs. These FIFO models are written out in Verilog and can be synthesized as well as simulated. A testbench is generated for this purpose.

For more information about the SYNCore FIFO compiler, refer to the following:

  • The built-in FIFO Compiler document, which you access from the SyncFifo Info button.
  • The Synplicity tool synthesis tool online help, where you can access information for the following from the online help Contents:
