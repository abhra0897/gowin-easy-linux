syn_direct_set Attribute

The syn_direct_set attribute controls the assignment of a net to the dedicated set pin of a synchronous storage element (flip-flop). Using this attribute, you can direct the mapper to only use a particular net as the set when the design has a conditional set on multiple candidates. This attribute can only be applied to separate OR logic and is valid for only one input of the set logic cone:

ValueDescription
0 Disables net assignment to the set pin of flip-flop.
1 Enables net assignment to the set pin of flip-flop.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
