syn_hier Attribute

The syn_hier attribute controls the amount of hierarchical transformation across boundaries on a module or component instance during optimization. You can apply syn_hier globally or to a module.

ValueDescription
soft The tool determines the best optimization across hierarchical boundaries.
firm Preserves the interface of the design unit. When there is cell packing across a boundary, the interface is changed.
hard Preserves the interface of the design unit and prevents most optimizations across hierarchical boundaries.
fixed Preserves the interface of the design unit with no exception. Prevents any optimizations from being performed across a hierarchical boundary.
remove Removes the level of hierarchy for the design unit. The hierarchy is reconstructed in the netlist and the Technology view schmeatics. This setting only affect synthesis optimization.
macro Preserves the interface and contents of the design with no exceptions.
flatten Flattens the hierarchy at all levels of the design. The hierarchy is reconstructed in the netlist and the Technology view schematics. This setting only affects synthesis optimization.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
