syn_direct_enable Attribute

The syn_direct_enable attribute controls the assignment of a clock enable net to the dedicated enable pin of a storage element (flip-flop). This attribute specifies the mapper to use a particular net as the only clock enable when the design has multiple clock-enable candidates.

ValueDescription
0 Disables net assignment to the clock enable pin.
1 Enables net assignment to the clock enable pin

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
