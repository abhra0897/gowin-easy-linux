syn_romstyle Attribute

The syn_romstyle attribute specifies the implementation to use for an inferred ROM. You can apply syn_romstyle globally, to a module, or to a ROM instance. To infer a ROM architecture, use case statements in your code. For the synthesis tool to implement a ROM, at least half of the available addresses in the case statement must be assigned a value.

ValueDescription
block_rom Maps inferred ROM to the appropriate vendor-specific memory, using the dedicated memory resources on the FPGA.
logic Maps inferred ROMs to registers.
select_rom Implements RAM using the distributed RAM resources.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
