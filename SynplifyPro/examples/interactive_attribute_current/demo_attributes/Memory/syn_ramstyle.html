syn_ramstyle Attribute

The syn_ramstyle attribute specifies the implementation to use for an inferred RAM. You can applysyn_ramstyle globally, to a module, or to a RAM instance.

ValueDescription
registers Maps inferred RAMs to registers. Use this setting to turn off RAM inference.
block_ram Maps inferred RAM to the appropriate vendor-specific memory, using the dedicated memory resources on the FPGA.
select_ram Implements RAM using the distributed RAM resources.
no_rw_check Disables the insertion of bypass logic around the RAM. The bypass logic is required to prevent possible mismatches when a design can simultaneously read and write to the same RAM location. For designs that never read and write to the same location simultaneously, specify this value to minimize overhead logic.
