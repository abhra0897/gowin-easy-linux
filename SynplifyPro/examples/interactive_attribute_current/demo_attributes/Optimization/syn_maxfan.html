syn_max_fan Attribute

The syn_max_fan attribute guides the fanout for an individual input port, net, or register output. The attribute value functions as a hard limit when it is attached to nets, ports, primitive instances, or registers.

ValueDescription
integer Overrides the default fanout guide value. The default fanout value varies with the technology.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
