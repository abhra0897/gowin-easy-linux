syn_pipeline Attribute

The syn_pipeline attribute allows registers that are outputs of multipliers to be moved to improve frequency. Depending on the criticality of the path, the output register can be either pushed into the multiplier or pushed back to the input side.

ValueDescription
0 Disables pipelining.
1 Enables pipelining.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
