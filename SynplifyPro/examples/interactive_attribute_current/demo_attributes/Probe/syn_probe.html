syn_probe attribute

The syn_probe attribute inserts probe points for the testing and debugging of internal signals. The probed signals appear as ports at the top level of the design.

ValueDescription
1 Generates probe names according to the net name.
portName Specifies the probe port name. If the signal is a bus slice, the port name is also the portName value.
portname[ ] Specifies the probe port name assigned to portName and indexed to the net names.

For detailed information about this attribute, use the Help button in the attribute wizard or the online help in the synthesis tool.
