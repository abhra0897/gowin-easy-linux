Interactive Attribute Examples

The Attribute wizard lets you quickly select attributes and values and generate examples on the fly.

  1. Double-click the attribute from the list on the left to start the wizard.
  2. Select the attribute values and click Generate. The wizard creates a project with Verilog source code and constraint files for the attribute, with separate implementations for each attribute value selected.

    The wizard automatically runs all implementations and opens the Technology views to show the impact of the attribute settings. It also opens the Log Watch window to compare resource utilization and timing information across implementations.

For more information about the attribute click the Help button in the Attribute wizard or consult the online help.
