PCI Target

Information

Type: PCI Target
Vendor: GOWIN Semiconductor
Summary: The Gowin PCI Target Local Bus is a high performance 32bit bus with multiplexed address and data lines. The bus is intended for use as an interconnect mechanism between highly integrated peripheral controller components, peripheral add-in boards, and processor/memory systems.

Options

Option Description
Bus Definition
PCI Data Bus Size PCI Data Bus Width.
Local Data Bus Size Local Data Bus Width.
PCI Bus Speed The System Clock frequency of PCI BUS.
ID
Vendor ID The Vendor ID is used to identify the manufacturer of product. This register is 16-bit and read-only.
Device ID The Device ID is defined by the manufacturer. It is used to uniquely identify a particular product or model. This register is 16-bit, read-only.
Subsystem Vendor ID The Subsystem Vendor ID is used to further identify the manufacturer of the expansion board or subsystem. This register is 16-bit, read-only.
Subsystem ID The Subsystem ID is used to further identify the particular device. This register is 16-bit, read-only.
Revision ID The Revision ID is a device-specific field that is set using the REVISION_ID parameter. This register is 8-bit, read-only.
Class Code The Class Code is a register and is used to identify the generic functionality of a device. This register is 24-bit, read-only.
Options
Devsel Timing These bits indicate the slowest time for all device accesses to assert the pci_devsel_l signal except the configuration.
Fast Back to Back This is a read-only bit that indicates if the device is capable of handling fast back-to-back transactions.
Interrupt Pin It is used to indicate which of the four interrupts that the PCI IP core uses.
BARs
Number of BARs 0：No base region enable
1：BAR0 enable
2：BAR0、BAR1 enable
3：BAR0、BAR1、BAR2 enable
4：BAR0、BAR1、BAR2、BAR3 enable
5：BAR0、BAR1、BAR2、BAR3、BAR4 enable
6：BAR0、 BAR1、 BAR2、 BAR3、 BAR4 、BAR5 enable.
BAR0 To BAR5 Configuration Options
BAR width The size of BAR register. It is only 32 in size.
BAR Type IO Read/Write or Memory Read/Write.
BAR Address Space Size The size of BAR Address Space. The range of BAR Address Space Size is 16 – 2G bytes.
BAR Prefetching This option determines if the memory mapped by this BAR support prefetching operation.
