SPI MASTER

Information

Type: SPI MASTER
Vendor: GOWIN Semiconductor
Summary: Gowin SPI Master is a SPI Master controller with a synchronous SRAM interface.

Options

Option Description
Master 1： Master mode
Slave Number Specify the number of slaves that can be supported.
Shift Direction Specify the most significant bit (MSB) or least significant bit (LSB) of the priority transmission data：
0： Priority transfer MSB
1： Priority transfer LSB
Clock Phase Specify the clock phase of the SPI：
0： Data is valid on the first edge of SCLK
1： Data is valid on the second edge of SCLK
Clock Polarity Specify the clock polarity of the SPI：
0： SCLK is active high
1： SCLK is active low
Clkcnt Width Specify the range of the clock counter, the bit width must be wide enough to meet the data width of SCLK.
Clock Sel Specify the division factor required to generate SCLK by S_AXI_ACLK. SCLK can be calculated by the following formula:
SCLK=S_AXI_ACLK/(2*(CLOCK_SEL+1)
Delay Time Specify the delay time to wait before the first bit of data is transmitted after the SS_N signal is asserted. The delay time that is waited for can be calculated by the following formula:
Delay=DELAY_TIME*(SCLK period/2)
Interval Length Specify the number of SCLK cycles that the SS_N signal waits after the SPI transfer request.
