I3C DDR eXtension

Information

Type: I3C DDR eXtension
Vendor: GOWIN Semiconductor
Summary: Gowin IPCore MIPI I3C Controller IP is compliant with the latest I3C specification and delivers high bandwidth and configurability for integration of multiple sensors into mobile, automotive and IoT system-on-chips (SoCs). The support for in-band interrupts within the 2-wire interface provides significantly lower pin count, simplifying board design and reducing power and cost. The standard-based synchronous SRAM bus connects the IP to the rest of the SoC while the bus is connected to the register interface and internal cache FIFO, offering easy IP integration. This version supports DDR mode based on SDR.
