AHB to APB 32 Bridge

Information

Type: AHB to APB 32 Bridge
Vendor: GOWIN Semiconductor
Summary: The Gowin AHB to APB 32 Bridge IP is used for converting transactions between an AHB bus and an APB bus. The bridge is compliant with AMBA 4 specifications. It is a slave on the AHB bus and is the only master on the APB bus.

Options

Option Description
ADDR_WIDTH Define this macro to set the address width. Default address width is 32-bit
ADDR_DECODE_WIDTH Define ADDR_DECODE_WIDTH to identify the width of the APB address decoding logic
SLV_<N> Define SLV_<N> to enable nth APB slave ports, where n = 1~31
SLV<N>_SIZE Define SLV<n>_SIZE(n = 1~31) to identify the space size of the nth APB slave based on 24-bit mode and 32-bit mode
SLV<N>_OFFSET Define SLV<N>_OFFSET(n = 1~31) to identify the base address offset of the nth APB slave
