RAM Based Shift Register

Information

Type: RAM Based Shift Register
Vendor: GOWIN Semiconductor
Summary: Gowin RAM Based Shift Register core provides a very efficient multi-bit wide shift register for use in FIFO-like applications or as a delay line. Fixed-length shift registers and variable-length shift registers can be created.

Options

Option Description
Data Width
Width Specify the width of the data.
Ram Type
LUT Based Contains LUTs and logic implementation.
BSRAM Based Contains Block RAM implementation.
SSRAM Based Contains Distributed RAM implementation.
Shift Register Type
Fixed Shift The shift is fixed and is always equal to the number of shifts.
Variable Shift Lossy The shift is controlled by the Addr input. When the “Addr” changes, the current input data will appear at the output Addr cycles later.
Variable Shift LossLess The shift is controlled by the Addr input. When the “Addr” changes, the current output data is what was at the input Addr cycles earlier.
Number of Shifts Specify the number of shifts for a fixed-length shift register.
Max Number of Shifts Specify the maximum number of shifts for a variable length shift register. This will determine the size of the “Addr” input.
Output Register
Enable Output Register Specify that registers are used for the output data. This will produce an additional clock cycle delay for the output.
SET/CLEAR
SET When SET is high, the output data are high.
CLEAR When CLEAR is high, the output data are low.
