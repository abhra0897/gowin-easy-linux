Complex Multiplier

Information

Type: Complex Multiplier
Vendor: GOWIN Semiconductor
Summary: Based on the needs of complex multiplication width, the Gowin Complex Multiplier IP can be configured as 9x9, 18x18 or 36x36 mode. The Gowin Complex Multiplier IP which is configured as 9x9 mode contains 4 MULT9X9 and 2 ALU54D. The Gowin Complex Multiplier IP which is configured as 18x18 mode contains 2 MULTADDALU18X18. The Gowin Complex Multiplier IP which is configured as 36X36 mode contains 4 MULT36X36 and 2 ALU54D.

Options

Option Description
Data Options
Input Width Set the size of the input in the complex multiplication.
Output Width Size of the output. The output size is 2*mul+1.
Data Type Set the data format of the inputs as signed or unsigned.
Register Options
Reset Mode Set whether the reset mode is synchronous or asynchronous.
Enable Input Register Enable or disable register. If you choose Enable Input Register, the input data will go through one register.
Enable Output Register Enable or disable register. If you choose Enable Output Register, the output data will go through one register.
Enable Pipeline Register Enable or disable register. If you choose Enable Pipeline Register, the output data will go through one register.

Note: mul = (Input Width <= 9)?9:(( Input Width <= 18)?18:36);When Input Width <=9,mul=9;When 9< Input Width <=18,mul=18;When 18< Input Width <=26,mul=36.
