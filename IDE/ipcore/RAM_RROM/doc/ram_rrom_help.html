rROM

Information

Type: rROM
Vendor: GOWIN Semiconductor
Summary: B-SRAM can be configured as rROM. In this mode, the design supports different read data widths. It can be initialized by a memory initialization file, and also supports two read modes (bypass and pipeline).

In rROM mode, the primitives rROM and rROMX9 can be used. The primitives support multiple data widths and address depths (Depth x width): rROM--16Kx1, 8Kx2, 4Kx4, 2Kx8, 1Kx16, 512x32; rROMX9--2Kx9, 1Kx18, 512x36.

Option Description
